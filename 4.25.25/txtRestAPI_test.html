Confluence Text File Manager

This example shows how to integrate the Confluence Text File Manager into a Confluence page.

Note: In a real Confluence page, you would add this via the HTML macro.

How to Integrate into Confluence

To add this Text File Manager to your Confluence page:

  1. Edit your Confluence page
  2. Add an HTML macro
  3. Paste the following code into the HTML macro:
<div id="confluence-text-file-manager"></div>

<script>
// Configuration
const textFileManagerConfig = {
  confluenceUrl: 'https://your-confluence-instance.atlassian.net',
  apiToken: 'YOUR_API_TOKEN',
  pageId: 'YOUR_PAGE_ID',
  containerId: 'confluence-text-file-manager'
};

// Function to load the script
function loadTextFileManagerScript() {
  const script = document.createElement('script');
  script.src = 'https://path-to-your-hosted-script/confluence-text-file-manager.js';
  script.onload = function() {
    // Initialize the Text File Manager when the script is loaded
    initConfluenceTextFileManager(textFileManagerConfig);
  };
  document.head.appendChild(script);
}

// Load the script when the page is ready
document.addEventListener('DOMContentLoaded', loadTextFileManagerScript);
</script>

Hosting the Script

You have several options for hosting the JavaScript file:

  1. Attach to the Confluence page: Upload the JS file as an attachment to your Confluence page, then reference it using the attachment URL.
  2. Host on a web server: Host the file on your organization's web server.
  3. Use a CDN: Host the file on a content delivery network.
Security Note: Make sure your API token has appropriate permissions and is securely stored. For production use, consider implementing proper authentication handling rather than embedding the token directly in the script.

Alternative: Embedding the Script

Alternatively, you can embed the entire script in your Confluence page's HTML macro:

<div id="confluence-text-file-manager"></div>

<script>
// Paste the entire contents of confluence-text-file-manager.js here

// Then initialize:
document.addEventListener('DOMContentLoaded', function() {
  const textFileManager = initConfluenceTextFileManager({
    confluenceUrl: 'https://your-confluence-instance.atlassian.net',
    apiToken: 'YOUR_API_TOKEN',
    pageId: 'YOUR_PAGE_ID',
    containerId: 'confluence-text-file-manager'
  });
});
</script>
