EnableAI

This bookmarklet gives product owners a simple way to generate realistic accessibility personas and user stories that humanize WCAG guidelines with specific challenges, technologies, and contexts. By selecting a UI feature, product owners instantly receive structured prompts with relevant personas (like "Maria with macular degeneration who uses screen magnifiers") and detailed user journeys that translate technical requirements into authentic human experiences. This allows product owners to effectively communicate accessibility needs to their teams without requiring deep technical expertise in accessibility standards.

  • Creates detailed accessibility personas with disabilities, assistive technologies, and real-world challenges
  • Generates human-centered user stories that translate technical WCAG guidelines into authentic experiences
  • Helps product owners communicate accessibility requirements without needing technical expertise
  • Organizes content by UI feature (navigation, forms, multimedia, etc.) for targeted implementation
  • Combines persona context, technical considerations, and WCAG guidelines in one structured prompt
  • Enables empathy-driven product design through realistic human scenarios
  • Works as a simple bookmarklet that floats on any webpage

LLM Considerations

allyLLM_v1.0.js

This JavaScript bookmarklet creates an accessibility interaction guide panel for ChatGPT that generates prompts tailored to various disabilities and assistive technologies. When activated on ChatGPT, it provides a floating interface where users can select specific accessibility needs and generate appropriate prompts that help ChatGPT deliver more accessible responses.

Key Features:

  • Supports multiple disability types (visual, hearing, physical, cognitive, communication, learning)
  • Includes specific assistive technology options for each disability type
  • Generates structured prompts with specific accessibility considerations
  • Offers direct prompt injection or clipboard copying
  • Customizable interface with light/dark themes and panel styles
  • Includes detailed interaction patterns and considerations for each assistive technology

Accomm

This is a ChatGPT-specific JavaScript bookmarklet called "Accommodations Assistant" that creates a floating panel interface allowing users to generate structured prompts about disability accommodations. It helps users select disability types, specify environments (workplace or public space), and automatically formats comprehensive accommodation assessment requests that can be directly injected into ChatGPT or copied to clipboard.

Key features:

  • Comprehensive database of disabilities categorized by type (visual, hearing, physical, cognitive, etc.)
  • Contextual information for both workplace (employee-facing) and public space (customer-facing) environments
  • Toggle between light/dark themes and different panel styles
  • One-click injection of generated prompts directly into ChatGPT
  • Accessible design with consideration for screen readers and keyboard navigation
