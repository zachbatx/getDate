Usage Tips

  • Size: Adjust the width and height attributes to scale the icons
  • Color: Use CSS color property or change currentColor to any valid color value
  • Inline Usage: You can embed these directly in your HTML
  • CSS Styling: Apply classes and style them like any other HTML element
